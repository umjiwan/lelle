### Commands
`도움말`: 명령어 리스트, 명령어의 세부 사용법을 알려줍니다.<br>
`주식`: 원하는 주식에 대한 정보를 알려줍니다.<br>
`원주율`: 원주율을 알려주거나 입력한 원주율이 일치하는지 검토해줍니다.<br>
`유러`: 원하는 문장을 유러로 암호화 해주거나 유러를 원래 문장으로 복호화 해줍니다.<br>
`핑`: 봇 지연시간을 ms 단위로 알려줍니다.
